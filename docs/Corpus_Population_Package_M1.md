## 4. Source and Evidence Notes

All records in this package are anchored to the GlyLens source and evidence registry design.

- `FAO/WHO Glycemic Index tables` is treated as academic, evidence level A, trust score 88-94.
- `USDA FoodData Central` is treated as government, evidence A/B, trust score 90-95.
- `India Food Composition Tables (IFCT)` is treated as government, evidence A/B, trust score 88-93.
- `National Institute of Nutrition India` is treated as government, evidence A/B, trust score 88-93.
- Manufacturer nutrition labels are treated as industry, evidence level B or C depending on supporting reference quality.
