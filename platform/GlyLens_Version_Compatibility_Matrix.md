# GlyLens Version Compatibility Matrix v1

_Last Updated: 2026-06-26_  
_Build Program: 1.3_  
_Status: CANONICAL_  
_Owner: Platform Engineering_

---

## Purpose

Define **one officially supported toolchain matrix** for GlyLens Windows development, derived from compatibility between Flutter stable, Dart, Android Gradle Plugin (AGP), Gradle, JDK, and Android SDK — not arbitrary version pins.

---

## Recommended Matrix (Official)

| Layer | Version | Source / Rationale |
|-------|---------|-------------------|
| **Flutter** | **3.27.4** (stable) | Matches CI pin (`.github/workflows/flutter_ci.yml`); satisfies `pubspec.yaml` `flutter: ">=3.24.0"`; bundles Dart 3.6.2 and `intl 0.20.2` via `flutter_localizations` |
| **Dart** | **3.6.2** | Bundled with Flutter 3.27.4; satisfies `sdk: ^3.5.0` |
| **Android Studio** | **2024.3.1.16** (Meerkat) | Supports AGP 3.2–8.10 per [Android Studio–AGP compatibility](https://developer.android.com/build/releases/about-agp); stable channel; includes SDK Manager + Emulator |
| **Android Gradle Plugin** | **8.7.3** | Generated by `flutter create` with Flutter 3.27.x template; AGP 8.7 requires Gradle ≥8.9 |
| **Gradle Wrapper** | **8.10.2** | AGP 8.7–8.8 compatible per [AGP–Gradle table](https://developer.android.com/build/releases/about-agp); Flutter 3.27 default |
| **JDK (build)** | **17.0.13** (Eclipse Temurin) | AGP 8.x minimum JDK 17; Flutter/Android tooling tested on 17; avoids JDK 21 path conflicts on this workstation |
| **compileSdk** | **35** | App compile target; Flutter 3.27 template default |
| **targetSdk** | **35** | Align with compileSdk for new projects |
| **minSdk** | **21** | Flutter default; covers ~99% devices |
| **Android SDK Platform (install)** | **≥ API 35** | e.g. `android-35`, `android-36.1`; newer platforms are fine |
| **Android SDK Build-Tools** | **37.0.0** | Latest stable; backward-compatible with compileSdk 35 |
| **Emulator system image** | **API 35+** | Match installed platform (e.g. API 36 if `android-36.1` installed) |
| **NDK** | **27.2.12479018** | Flutter plugin default (when required) |
| **Kotlin** | **2.0.21** | Flutter 3.27 Android template default |

## SDK install vs app compile target

| Concept | GlyLens value | Notes |
|---------|---------------|-------|
| **compileSdk / targetSdk** (in `android/app/build.gradle`) | **35** | What the app is built against; stays at 35 until Flutter/CI matrix is bumped |
| **SDK Platform package** (SDK Manager) | **≥ 35** | e.g. `android-36.1` on your machine — newer is fine |
| **Build-Tools** (SDK Manager) | **37.0.0** | Version number is independent of API level; 37.0.0 works with compileSdk 35 |

---

## Why This Matrix (Not Newer AGP 9 / Flutter 3.32+)

| Alternative considered | Rejected because |
|------------------------|------------------|
| Flutter 3.32+ / latest stable | Would require CI matrix re-validation, EBOM update, and potential AGP 9 migration |
| AGP 9.0 + Gradle 9.1 | Requires JDK 17+, new DSL defaults, Studio Panda+; out of scope for BP1.3 stand-up |
| JDK 21 (currently on machine) | Works with AGP 8.7+ in many cases, but `flutter doctor` and Gradle JDK settings are cleaner with **17**; machine has 21 without `JAVA_HOME` |
| Ubuntu 24.04 WSL only | Machine has Ubuntu 26.04; Docker supports both; not blocking |

**Decision:** Flutter **3.27.4** + **AGP 8.7.3** + **Gradle 8.10.2** + **JDK 17** is the **lowest-risk, CI-aligned** matrix for the next 18 months.

## Version Verification Commands

After stand-up, all of the following must succeed:

```powershell
flutter --version          # 3.27.4 • Dart 3.6.2
java -version              # openjdk 17.x
cd android; .\gradlew.bat --version   # Gradle 8.10.2
flutter doctor -v          # Android toolchain ✓
```

## Upgrade Policy

| Component | When to upgrade |
|-----------|-----------------|
| Flutter patch (3.27.x) | After CI green; quarterly EBOM review |
| Flutter minor (3.28+) | Platform Engineering review; re-run this matrix |
| AGP major (9.x) | Dedicated migration sprint; Studio Panda+ |
| JDK | Stay on 17 until AGP 9 migration or Flutter docs change floor |
